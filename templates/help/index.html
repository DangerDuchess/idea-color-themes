{% trans 'How to install a theme?' %}

{% blocktrans %} Select «File» → «Import Setting» from the main menu and follow the instructions. After the IDE restarted go to the Preferences, expand «Editor» → «Colors and fonts» tab and choose the installed theme. {% endblocktrans %}

{% trans 'Which IDEs are supported?' %}

{% blocktrans %} Themes on this site support fully all family of JetBrains' IDEs: InteliJ IDEA, PhpStorm, PyCharm, RubyMine, WebStorm and AppCode. {% endblocktrans %}

{% trans 'Which platforms are supported?' %}

{% blocktrans %} Windows, Mac OS and Linux are fully supported. {% endblocktrans %}
